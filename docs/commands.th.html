คำสั่ง

Clew มาพร้อมคำสั่ง slash ในตัว 80+ รายการ พร้อมสกิลที่โหลดแบบไดนามิกและคำสั่งปลั๊กอิน

เซสชันและบริบท

คำสั่งคำอธิบาย
/clearล้างประวัติการสนทนาและเพิ่มพื้นที่บริบท
/compactบีบอัดบริบทเซสชัน — ตัด log/trace, รวม, pruning
/contextแสดงภาพการใช้งานบริบทปัจจุบันเป็นกริดสี
/resumeกลับมาการสนทนาก่อนหน้า
/renameเปลี่ยนชื่อการสนทนาปัจจุบัน
/rewindคืนค่าโค้ดและ/หรือการสนทนาไปยังจุดก่อนหน้า
/sessionแสดง URL เซสชันระยะไกลและ QR code

โมเดลและผู้ให้บริการ

คำสั่งคำอธิบาย
/modelสลับโมเดลหรือผู้ให้บริการ — ดึงโมเดลสดจาก API ผู้ให้บริการ
/tasteเปิดเมนูการเรียนรู้ความชอบแบบโต้ตอบ
/effortตั้งค่าระดับความพยายามสำหรับการใช้เหตุผล (low, medium, high, max)
/costแสดงค่าใช้จ่ายและระยะเวลาทั้งหมดของเซสชันปัจจุบัน
/usageแสดงขีดจำกัดการใช้งานแผน
/statsแสดงสถิติการใช้งาน Clew

เครื่องมือและสิทธิ์

คำสั่งคำอธิบาย
/toolsแสดงเครื่องมือทั้งหมดพร้อมสถิติและค่าใช้จ่ายโทเค็น
/permissionsจัดการกฎสิทธิ์อนุญาตและปฏิเสธเครื่องมือ
/planเปิดโหมดวางแผนหรือดูแผนเซสชันปัจจุบัน
/capabilitiesแสดงความสามารถของระบบที่มี (git, tmux, browser, shells)

ส่วนขยาย (ปลั๊กอิน, MCP, สกิล)

คำสั่งคำอธิบาย
/pluginจัดการปลั๊กอิน — list, install, remove
/mcpจัดการเซิร์ฟเวอร์ MCP
/skillแสดงสกิลที่มีหรือรายละเอียดสกิล
/code-reviewรีวิวโค้ดที่เปลี่ยนแปลงหาบั๊ก

เอเจนต์

คำสั่งคำอธิบาย
/agentจัดการการทำงานของ Clew AI Agents
/agentsเปิดแดชบอร์ดเอเจนต์แบบโต้ตอบ
/daemonเปิดแผงควบคุม daemon อัตโนมัติ 24/7
/taskสร้างงานตามกำหนดการด้วยแบบฟอร์มโต้ตอบ
/loopรันคำสั่งซ้ำตามช่วงเวลา

ระบบและการตรวจสอบ

คำสั่งคำอธิบาย
/helpแสดงความช่วยเหลือและคำสั่งที่มี
/statusแสดงสถานะ Clew รวมถึงเวอร์ชัน, โมเดล, บัญชี
/doctorตรวจสอบการติดตั้งและการตั้งค่า Clew
/exitออกจาก REPL

การตั้งค่า

คำสั่งคำอธิบาย
/configเปิดแผงตั้งค่า
/themeเปลี่ยนธีม
/colorตั้งค่าสีแถบ prompt
/keybindingsเปิดหรือสร้างไฟล์ตั้งค่าคีย์ลัด

Git และรีวิวโค้ด

คำสั่งคำอธิบาย
/diffดูการเปลี่ยนแปลงที่ยังไม่ได้ commit
/commitstage และ commit การเปลี่ยนแปลง
/reviewรีวิวการเปลี่ยนแปลงโค้ด

การวิจัยและค้นหา

คำสั่งคำอธิบาย
/researchวิจัยเชิงลึกจากแหล่งที่มาในไฟล์ท้องถิ่น, wiki, และหน่วยความจำ
/memoryแก้ไขไฟล์หน่วยความจำ Claude
/memory-searchค้นหาหน่วยความจำด้วยการค้นหาเชิงความหมาย (ข้ามภาษา)

การโหลดคำสั่ง

คำสั่งถูกโหลดจากหลายแหล่งเมื่อเริ่มต้น:

  • ในตัว — คำสั่งหลักจาก src/commands/ (พร้อมใช้เสมอ)
  • สกิลที่มาพร้อม — สกิลที่มาพร้อมกับแอปพลิเคชัน
  • ไดเรกทอรีสกิล — ~/.claude/skills/ และ .claude/skills/
  • คำสั่งปลั๊กอิน — จากปลั๊กอินที่ติดตั้ง
  • MCP skills — จากเซิร์ฟเวอร์ MCP
