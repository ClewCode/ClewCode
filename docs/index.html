Ceph Code

A terminal-based AI coding assistant with multi-provider routing, extensible plugin architecture, and a comprehensive tool system for professional development workflows.

Getting Started

  1. Install — npm install -g cephcode
  2. Set API Key — export ANTHROPIC_API_KEY=sk-ant-...
  3. Launch — Run cephcode in your terminal
  4. Try commands — Type /model list, /status, or /help

Usage

Ceph Code works like a chat in your terminal. Type naturally to ask questions, request code changes, or run commands: 

# Start a session
cephcode

# Ask a question
"find all unused exports in src/"

# Run a command
/edit main.ts:42

# Switch model
/model deepseek-v4-flash

# Context management
/clear    # Fresh start
/compact  # Compress context
/resume   # Resume last session
