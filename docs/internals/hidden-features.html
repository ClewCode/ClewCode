Architecture Overview

Ceph Code is structured as a layered system with a terminal UI (React/Ink), a command/query processing layer, and a multi-provider AI execution engine. The architecture supports hot-swappable providers, plugin hooks at every stage of execution, and sandboxed tool execution.

Key Internals

  • Provider Manager — Dynamic provider selection and API key resolution
  • Adapter Layer — Normalizes non-Anthropic providers to a common interface
  • Tool Executor — Streaming tool execution with pre/post hooks
  • Session Bridge — Cross-session context persistence and recall
  • Code Index — Fuzzy code search across the entire codebase

Undocumented Slash Commands

These commands exist in the codebase but are not advertised in the help menu:

  • /ultraplan — Extended plan mode with deeper analysis
  • /bughunter — Automated bug hunting workflow
  • /heapdump — Capture Node.js heap snapshot
  • /perf-issue — Performance issue reporting
Note Hidden commands may require specific environment variables or feature flags. Check the source code for exact enablement conditions.
