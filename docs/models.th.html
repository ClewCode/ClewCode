โมเดล

Clew รองรับ 100+ โมเดลจาก 27 ผู้ให้บริการ เลือกโมเดลที่เหมาะกับแต่ละงาน — จากการใช้เหตุผลระดับสูงไปจนถึงการอนุมานในเครื่องที่รวดเร็ว

คู่มือการเลือกโมเดล

ลักษณะงานโมเดลแนะนำบริบทหมายเหตุ
สถาปัตยกรรมซับซ้อนและแก้ไขขนาดใหญ่claude-opus-4-7, claude-sonnet-4-6, gpt-5.5, deepseek-v4-pro1M โทเค็นการเรียกใช้เครื่องมือดีที่สุด, การใช้เหตุผลแข็งแกร่ง, บริบทใหญ่ที่สุด
รีวิวโค้ดและดีบักclaude-sonnet-4-6, gpt-5.5, gemini-3.1-pro1M-2M โทเค็นทำตามคำแนะนำดี, ผลลัพธ์เชื่อถือได้
ค้นหาและสรุปผลอย่างรวดเร็วclaude-haiku-4-5, deepseek-v4-flash, mistral-small-4, groq/llama-3.3-70b128K-200Kเร็ว, คุ้มค่า, หน่วงเวลาต่ำ
ออฟไลน์ / ไม่มีเครือข่ายollama/llama4:70b, ollama/llama3.3131K โทเค็นในเครื่องทั้งหมด, ไม่ต้องใช้เครือข่าย

การสลับโมเดล

/model                          # ตัวเลือกแบบโต้ตอบ
/model list                     # แสดงโมเดลทั้งหมดจากผู้ให้บริการที่ใช้งาน
/model claude-sonnet-4-7        # สลับด้วย ID เต็ม
/model sonnet                   # สลับด้วย alias
--model opus                    # CLI flag ตอนเริ่ม

CLI Flags สำหรับโมเดล

Flagคำอธิบาย
--model <name>ตั้งค่าโมเดลตอนเริ่ม
--effort <level>ตั้งค่าระดับการใช้เหตุผล: low, medium, high, max
--thinking <mode>โหมดการคิด: enabled, adaptive, disabled
--fallback-model <model>โมเดลสำรองเมื่อโมเดลหลักโอเวอร์โหลด
