Quick Start

Get Clew Code running in under a minute.

1. Install

npm install -g clew-code

Requires Bun 1.3+, Node.js 18+, Git, and one provider API key.

2. Run

cd my-project
clew

On first launch you'll be prompted to select a provider.

3. Configure a Provider

Inside the REPL, pick your provider and model:

❯ /model openai
❯ /model gpt-5.5

❯ /model deepseek-v4-pro

❯ /model ollama/llama3.3

Or set environment variables before starting:

export ANTHROPIC_API_KEY=sk-ant-...
export OPENAI_API_KEY=sk-...
export GOOGLE_API_KEY=...
export DEEPSEEK_API_KEY=...

4. Try Commands

❯ /help           # list everything
❯ /status         # current provider, model, context info
❯ /mesh discover  # find other Clew instances on LAN
❯ /mcp list       # connected MCP servers

One-Shot Mode

Pipe-friendly single-use mode:

clew -p "summarize CHANGELOG.md"

Resume Last Session

clew --resume last
