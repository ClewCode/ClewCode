Switching Providers

Use the /model command inside the REPL:

❯ /model openai
❯ /model gpt-5.5

❯ /model deepseek-v4-flash

❯ /model ollama/llama3.3

List Available Models

❯ /model list

Shows live model lists from provider APIs with capability tags:
[200K ctx, vision, tools, reason, free]

Reasoning Effort

Some providers expose reasoningEffort (NVIDIA, DeepSeek, OpenRouter, etc.). Control it with:

❯ /effort high
