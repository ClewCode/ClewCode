/tools

ระบบปลั๊กอิน

ขยายด้วยคำสั่ง, เอเจนต์, สกิล, ฮุค และ เซิร์ฟเวอร์ MCP ระบบฮุคก่อน/หลังเครื่องมือสำหรับระบบอัตโนมัติ

/plugin /skill

มุมมองเอเจนต์

แดชบอร์ดหลายเซสชันพร้อม daemon ผู้ดูแล, งานพื้นหลัง, การสื่อสาร IPC, การแยก worktree, คอลัมน์ PR, autocomplete และคีย์ลัด

/agents

เชลล์พื้นหลัง

รันคำสั่งยาวเป็นงานเอเจนต์พื้นหลังด้วย !bg <command> สถานะสด, รหัสออก, การจับ stderr และการรวมแดชบอร์ด

!bg

โหมดอัตโนมัติ 24/7

daemon ถาวรพร้อมคิวงาน, การจัดตารางอัตโนมัติ, การตรวจสอบสุขภาพ, การเริ่มใหม่อัตโนมัติ และการจัดระเบียบเวิร์กเกอร์

 /daemon /task

หน่วยความจำอัจฉริยะ

การค้นหาเชิงความหมายข้ามภาษา, การเรียกคืนแบบผสม (local + LLM), การบันทึกอัตโนมัติ และการซิงค์ทีม

 /memory

โหมด Bridge

การทำงานร่วมกันระยะไกลผ่าน WebSocket, การแชร์เซสชัน, การเริ่มต้นทีม และไคลเอนต์มือถือ/เว็บ

Bridge

KiloCompact

การบีบอัดบริบทในเครื่อง: การตัด log/trace, การรวมสถานะที่ล้มเหลว, การตัดรอบการสนทนาตามความหมาย

/compact

โหมดสิทธิ์

ค่าเริ่มต้น, อัตโนมัติ, แผน, YOLO Lite, YOLO, YOLO MAX — ควบคุมความปลอดภัยในการใช้เครื่องมือแบบละเอียด

/permissions

การเรียนรู้ความชอบ

ระบบ Taste ที่เรียนรู้สไตล์การเขียนโค้ดของคุณจากสัญญาณ accept/reject/edit เมนูแบบโต้ตอบ, การตรวจสอบการแก้ไข, การโหลดค่าตั้งค่าใหม่ทันที

/taste

เริ่มต้นใช้งาน

จากศูนย์ถึงการเขียนโค้ดในไม่ถึงนาที

bash 
$ npm install -g clew-code
$ export ANTHROPIC_API_KEY=sk-ant-...
$ clew

# ถามคำถาม
> "ค้นหา unused exports ทั้งหมดใน src/"

# สลับโมเดลทันที
> /model deepseek-v4-flash

# จัดการบริบท
> /clear    # เริ่มใหม่
> /compact # บีบอัดบริบท
> /resume  # กลับมาเซสชันล่าสุด

อัปเดต v0.2.4

  • ระบบ Taste — ระบบเรียนรู้ความชอบในเครื่อง เมนู /taste แบบโต้ตอบ แถบสถานะ, โหลดค่าตั้งค่าใหม่ทันที
  • ตัวเลือกโมเดลแบ่งตามผู้ให้บริการ — /model ตอนนี้แสดงโมเดลทั้งหมดจากทุกผู้ให้บริการ
  • บันทึกผู้ให้บริการอัตโนมัติ — ผู้ให้บริการและโมเดลล่าสุดบันทึกใน provider.json
  • เพิ่มความปลอดภัย — กฎ PowerShell, การป้องกัน tool call ผิดรูปแบบ, Guardian ตรวจสอบอัตโนมัติ
  • Bridge v2 — ควบคุมระยะไกลผ่าน WebSocket, relay client, REPL bridging
  • คำสั่ง PR — /pr create, list, view, review, merge, status
