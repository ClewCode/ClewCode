Commands

Clew Code provides 30+ slash commands for managing providers, sessions, mesh networking, MCP, plugins, and more.

Session Commands

CommandDescription
/statusShow current provider, model, session, and context info
/doctorRun system diagnostics
/contextShow active context usage breakdown
/compactCompress conversation history (multi-pass compaction)
/clearClear conversation or caches
/exitExit the REPL
/helpList all available commands
/resumeResume a previous session

Provider & Model Commands

CommandDescription
/modelSwitch provider or model. Use /model list to see available models
/effortSet reasoning effort (low/medium/high) for supported providers
/fastToggle fast/turbo mode

Mesh & Networking Commands

CommandDescription
/mesh discoverFind other Clew instances on the LAN
/mesh shareShare this instance as a worker
/mesh run <peer> <cmd>Run a command on a remote peer
/mesh send <peer> <msg>Send a message to a peer
/mesh listList discovered peers
/mesh info <peer>Show peer details
/mesh helpMesh command reference

MCP Commands

CommandDescription
/mcp listList connected MCP servers and tools
/mcp addAdd an MCP server configuration
/mcp removeRemove an MCP server
